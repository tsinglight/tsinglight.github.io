添加和删除节点

添加分成3步：

  1. 创建元素
  2. 创建节点追加到元素上
  3. 向已有的元素追加创建的新元素

删除节点有两步:

  1. 找到其父元素
  2. 删除父元素的子元素（即要删除的元素）

JavaScript 表单验证

JavaScript 可用来在数据被送往服务器前对 HTML 表单中的这些输入数据进行验证。

被 JavaScript 验证的这些典型的表单数据有：

  • 用户是否已填写表单中的必填项目？
  • 用户输入的邮件地址是否合法？
  • 用户是否已输入合法的日期？
  • 用户是否在数据域 (numeric field) 中输入了文本？
