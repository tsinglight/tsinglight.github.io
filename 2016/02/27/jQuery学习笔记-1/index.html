jQuery自定义动画

jquery animate()方法用于创建自定义动画

params参数定义形成动画的CSS属性，必须项
speed参数规定动画效果的时间长度，可选
callback参数规定动画完成后执行的函数名称，可选。

jQuery animate() 操作多个属性
jQuery animate() 使用相对值
jQuery animate() 使用队列功能
jQuery stop()方法，用于在动画或者效果完成之前对它们进行停止,适用于所有jQuery效果函数

通过jQuery，您可以把动作/方法连接起来。
Chaining 允许我们在一条语句中允许多个jQuery方法（在相同的元素上）。

jQuery 尺寸
  1. width()和height()方法
    width()方法设置或返回元素的宽度（不包括内边距、边框或者外边距）
    height()方法设置或返回元素的高度(不包括内边距边框或外边距)

  1. innerWidth()和innerHeight()方法用于创建自定义动画
    innerWidth()返回元素的宽度（包括内边距）
    innerHeight()方法返回元素的高度（包括内边距）

  2. outerWidth()和outerHeight()方法
    outerWidth()返回元素的宽度（包括内边距和边框）
    outerHeight()返回元素的高度（包括内边距和边框）

outerWidth(true)方法返回元素的宽度（包括内边距、边框和外边距）
outerHeight(true)方法返回元素的高度（包括内边距、边框和外边距）

jQuery 遍历-向上

jQuery parent()方法
该方法返回被选元素的直接父元素，只会向上一级对DOM树进行遍历

jQuery parents()方法

jQuery parentsUntil()方法

jQuery 遍历-向下

jQuery children()方法
返回被选择元素的直接子元素

jQuery find() 方法
返回被选元素的后代元素，一直向下直接到最后一个后代

jQuery 遍历-同胞

jQuery siblings()方法

jQuery next()方法
该方法返回被选元素的下一个同胞元素，只返回一个元素

jQuery nextAll()方法
nextAll()方法返回被选元素的所有跟随的同胞元素

jQuery nextUntil()方法
nextUntil()返回介于两个给定参数之间的所有跟随的同胞元素

jQuery 遍历-过滤

first()、last()、eq()、fliter()、not()

filter() 方法匹配所有符合标准的元素 而not()方法则相反

jQuery-AJAX

AJAX = 异步JavaScript 和 XML (Asynchronous JavaScript and XML)

在不重载整个网页的情况下，AJAX通过后台加载数据并在网页上显示

jQuery load() 方法

load()方法从服务器加载数据，并且把返回的数据放入被选元素中

URL:必须，规定了希望加载的URL
data:可选，规定与请求一起发送的查询字符串键/值对集合
callback:可选，load()方法完成后所执行的函数名称
回调函数可以设置不同的参数:
responseTxt - 包含调用成功时的结果内容
statusTXT - 包含调用的状态
xhr - 包含 XMLHttpRequest 对象
