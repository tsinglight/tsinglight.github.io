添加和删除节点

添加分成3步：

  1. 创建元素
  2. 创建节点追加到元素上
  3. 向已有的元素追加创建的新元素

删除节点有两步:

  1. 找到其父元素
  2. 删除父元素的子元素（即要删除的元素）
